
# ___Training Service API___

##  Версии
* JDK 25
* Gradle 9.2.1
* SpringBoot 3.3.5

## Основной стек зависимостей
* Spring Security
* Spring Web
* Spring Data JPA
* Postgresql
* Redis
* JUnit

## Описание
    REST API server для обучния на курсах и выполнения проектов студентами.

## Запуск приложения
* > Запустить приложение с помощью среды разработки или поднять с помощью docker-compose.yaml
* > Проверить запросы с помощью request-test.http
